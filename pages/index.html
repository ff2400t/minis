Mini Tools Dashboard

A collection of fast, browser-based utilities for document and data processing.

  • PDF Data Extractor

    Extract structured data from GST, Income Tax, and Bank statements using predefined or custom regex parsers.

  • RegEx Toolbox

    Interactive playground for building and testing regular expressions with real-time highlighting and group extraction.

  • Excel Sheet Combiner

    Merge multiple Excel files into a single workbook, preserving all sheets and content.

  • PDF OCR & Overlay

    Convert scanned PDFs or images into searchable PDF files with invisible text overlays.

  • PDF Table Extractor

    Recover tabular data from searchable PDFs with automatic column detection and manual adjustment.

  • Excel Consolidator

    Consolidate rows from multiple Excel sheets across different files based on a target header anchor.

  • PDF Modifier

    Quickly rotate, reorder, or remove pages from your PDF documents.

  • Excel Formula Studio

    Interactive playground for building and testing complex Excel formulas.

  • Kanata Visualizer

    Visualize and debug Kanata keyboard remapping configurations.

  • Docx Templater

    Generate Word documents from templates by filling in data markers.

Built for privacy & speed. All processing happens locally in your browser.
